## 销售职员功能模块

销售职员主要负责客户维护、销售合同管理、发货跟踪、开票和收款等工作。

### 1. 客户管理

**功能说明**：管理销售客户档案信息

**主要功能**：
- 查看客户列表：展示所有客户信息（名称、联系人、电话、邮箱、地区等）
- 搜索客户：按名称、地区进行搜索
- 新增客户：填写客户详细信息（名称、联系人、电话、邮箱、地址、行业、地区、开户银行、银行账号等）
- 编辑客户：修改客户信息
- 删除客户：删除客户档案
- 查看详情：查看客户详细信息及关联的销售合同列表

### 2. 销售合同管理

**功能说明**：管理销售合同的全生命周期

**主要功能**：
- 查看合同列表：展示所有销售合同（合同编号、品名、客户、签订日期、总金额、执行进度、收款进度、开票进度、状态）
- 新增合同：选择客户、填写品名、单价、数量，系统自动计算总金额
- 编辑合同：修改合同信息
- 删除合同：删除合同记录
- 查看详情：查看合同基本信息、关联客户、发货批次、发票列表、收款记录
- 进度跟踪：在详情页可点击到货进度、收款进度、开票进度，快速跳转到对应新增界面
- 合同附件：上传和下载合同附件（PDF、图片等）

**自动通知**：销售合同创建后，系统会自动通知采购人员

**注意**：进入销售合同详情后可以点击对应的 到货/收款/开票 进度模块，自动跳转到新增页面

 ![新增到货弹窗](./使用指南图片/屏幕截图%202026-03-11%20143142.png)


### 3. 销售运输管理

**功能说明**：记录客户到货批次

**主要功能**：
- 查看运输列表：展示所有运输记录（运输合同号、品名、合同编号、物流公司、到货日期、数量、收货地址）
- 新增到货记录：选择关联的销售合同、填写运输合同号、到货日期、到货数量、物流公司、收货地址
- 编辑运输信息：修改运输记录
- 删除记录：删除运输记录
- 附件管理：上传和下载物流单据附件

**注意**：此模块只需要记录客户到货进度，发货以及运输物流由采购人员负责

### 4. 销售发票管理

**功能说明**：管理销售发票开具记录

**主要功能**：
- 查看发票列表：展示所有销售发票（发票号码、品名、合同编号、开票日期、发票金额）
- 新增发票：选择关联的销售合同、填写发票号码、发票类型、产品数量、发票金额、开票日期
- 自动计算：填写产品数量后，系统会根据关联销售合同的产品单价自动计算发票金额
- 编辑发票：修改发票信息
- 删除发票：删除发票记录
- 附件管理：上传和下载发票附件

**自动更新**：开票后，合同的开票进度会自动更新

### 5. 销售收款管理

**功能说明**：管理销售回款登记和跟踪

**主要功能**：
- 查看收款列表：展示所有收款记录（合同编号、品名、收款日期、收款金额、产品数量、收款方式）
- 新增收款：选择关联的销售合同、填写收款金额、产品数量、收款日期、收款方式、收款账户
- 自动计算：填写产品数量后，系统会根据关联销售合同的产品单价自动计算收款金额
- 编辑收款：修改收款信息
- 删除收款：删除收款记录
- 附件管理：上传和下载收款凭证附件

**自动更新**：收款后，合同的收款进度和欠款金额会自动更新

### 6. 通知中心

**功能说明**：接收采购创建合同的通知

**主要功能**：
- 查看通知列表：显示所有收到的通知（类型、标题、时间、已读/未读状态）
- 查看详情：点击通知查看详细内容
- 标记已读：将通知标记为已读状态
- 未读数量提示：左侧导航栏显示未读通知数量

**触发场景**：
- 当采购人员新建采购合同时（未关联销售合同），系统会自动发送通知给销售人员

## 采购职员功能模块

采购职员主要负责供应商维护、采购合同管理、到货跟踪、收票和付款等工作。

### 1. 供应商管理

**功能说明**：管理采购供应商档案信息

**主要功能**：
- 查看供应商列表：展示所有供应商信息（名称、联系人、电话、邮箱、地区等）
- 搜索供应商：按名称、地区进行搜索
- 新增供应商：填写供应商详细信息

- 删除供应商：删除供应商档案
- 查看详情：查看供应商详细信息及关联的采购合同列表

### 2. 采购合同管理

**功能说明**：管理采购合同的全生命周期

**主要功能**：

- 合同附件：上传和下载合同附件

**注意**：进入采购合同详情后可以点击对应的 到货/收款/开票 进度模块，自动跳转到新增页面

### 3. 采购运输管理

**功能说明**：管理采购到货批次和物流跟踪信息

**主要功能**：
- 查看运输列表：展示所有运输记录
- 新增运输记录：选择关联的采购合同、可选关联销售合同、填写运输合同号、发货日期、到货数量、物流公司、发货地址、收货地址
- 中专仓库选项：可选择是否有中专站（是/否）
  - 选择"否"：运费2、中专仓库等相关字段无需填写
  - 选择"是"：需要填写运费2、中专仓库、运费2状态、运费2日期、发票2状态

- 运费管理：填写运费1（必填）、运费2（可选）、选择运费状态
- 编辑运输信息：修改运输记录
- 删除记录：删除运输记录
- 附件管理：上传和下载物流单据附件

**注意**：物流运输中途的各种费用都由采购人员负责填写记录

## 管理功能模块

经理主要负责全局合同查看、进度跟踪、销售与采购关联对比分析以及数据报表统计。

### 1. 合同总览

**功能说明**：全局查看所有销售和采购合同

**主要功能**：
- 查看销售合同列表：查看所有销售合同信息
- 查看采购合同列表：查看所有采购合同信息
- 销售合同详情：查看销售合同的详细信息，包括客户信息、发货批次（显示为客户到货批次）、发票记录、收款记录
- 采购合同详情：查看采购合同的详细信息，包括供应商信息、运输批次（显示为采购运输批次）、收票记录、付款记录

### 2. 进度跟踪

**功能说明**：全局监控所有合同的执行进度

**主要功能**：
- 销售合同进度：查看发货（到货）、收款、开票进度
- 采购合同进度：查看到货（发货）、收票、付款进度
- 筛选搜索：按合同编号、客户/供应商名称等条件筛选
- 查看详情：点击任意合同查看详细信息

### 3. 关联对比

**功能说明**：对比分析销售与采购的关联进度和利润

**主要功能**：
- 选择合同：选择一个销售合同或采购合同进行对比
- 关联信息查看：查看所选合同关联的对方合同详情
- 进度对比：
  - 客户到货 vs 采购发货
  - 收款 vs 付款
  - 开票 vs 收票
  - 支持多合同对比：当一个销售合同对应多个采购合同（或反之）时，显示多条进度条
- 利润分析：
  - 销售单价/总金额 vs 采购单价/总金额
  - 营业利润 = 销售总金额 - 采购总金额 - 运费1 - 运费2（如有）- 杂费（如有）
  - 净利润 = 销售总金额 - 采购总金额 - 税 - 运费及杂费
  - 税计算公式：税 = (销售总价 × 1.13 - 采购总价 × 1.13) × 0.1878

**注意**：利润计算需要销售合同总金额等于对应采购合同总金额之和才能进行

### 4. 数据报表

**功能说明**：生成销售采购财务报表，支持按时间段筛选和导出

**主要功能**：
- 时间段选择：按月份选择报表时间范围（如1-2月、2-3月）
- 报表内容：包含采购合同编号、签订日期、产品名称、供应商名称、产品数量、采购单价、采购总价、销售合同号、签订日期、客户名称、产品数量、销售单价、销售总价、销售税额、运费、杂费、客户到货时间、营业利润、净利润
- 报表统计：显示总计行（销售总价总计、采购总价总计、销售税额总计、运费总计、杂费总计、营业利润总计、净利润总计）
- 报表筛选：只显示已完成的销售合同和采购合同
- 多合同对应处理：支持一个销售合同对应多个采购合同，或一个采购合同对应多个销售合同的情况
- 导出功能：支持将报表导出为Excel文件

**计算公式**：
- 营业利润 = 销售总金额 - 采购总金额 - 运费1 - 运费2（如有）- 杂费（如有）
- 净利润 = 销售总金额 - 采购总金额 - 税 - 运费及杂费
- 税计算公式：税 = (销售总价 × 1.13 - 采购总价 × 1.13) × 0.1878

## 业务流程说明

### 流程一：销售先签订合同

```
销售流程：客户管理 → 签订销售合同 → 客户到货 → 开具发票 → 登记收款
                                    ↓
                                系统通知采购

采购流程：接收通知 → 查看销售合同 → 新建关联采购合同 → 采购运输 → 收票 → 付款
```

### 流程二：采购先签订合同

```
采购流程：供应商管理 → 签订采购合同 → 系统通知销售 → 采购运输 → 收票 → 付款
                                    ↓
                                系统通知采购

销售流程：接收通知 → 查看采购合同 → 签订销售合同 → 客户到货 → 开具发票 → 登记收款
```

## 注意事项

1. **合同关联**：销售合同和采购合同，先创建的会通知另一方，另一方就需要关联对应的合同（例如：销售先创建合同，系统通知采购，采购创建合同时需要关联对应销售合同）
2. **自动计算**：系统会自动计算合同的各种进度百分比和金额
3. **自动通知**：当创建合同且未关联对方合同时，系统会自动发送通知
4. **附件上传**：各模块都支持附件上传功能，可以上传相关凭证文件
5. **进度跳转**：在合同详情页面，可点击进度链接快速跳转到对应新增界面，且会自动填充关联信息
