# 功能修改文档：添加合同负责人字段

## 1. 需求概述

在销售合同和采购合同中添加「销售负责人」和「采购负责人」字段，用于记录每笔合同的负责人信息。

## 2. 后端字段信息

| 合同类型 | 字段名 | 字段类型 | 说明 |
|---------|--------|---------|------|
| 销售合同 | `sales_manager` | text | 销售负责人 |
| 采购合同 | `purchasing_manager` | text | 采购负责人 |

> **注意**：后端字段已在 PocketBase 中创建完成，前端需要添加对应的展示和编辑功能。

## 3. 前端修改清单

### 3.1 类型定义修改

#### 3.1.1 销售合同类型 (`src/types/sales-contract.ts`)

在 `SalesContract` 接口中添加 `sales_manager` 字段：

```typescript

  formData.append('sales_manager', data.sales_manager || '');
}
```

## 6. 测试要点

1. **销售合同新增**：填写销售负责人后保存，检查数据库字段是否正确
2. **销售合同编辑**：修改销售负责人，检查是否正确更新
3. **销售合同详情**：查看详情页是否正确显示销售负责人
4. **采购合同新增**：填写采购负责人后保存，检查数据库字段是否正确
5. **采购合同编辑**：修改采购负责人，检查是否正确更新
6. **采购合同详情**：查看详情页是否正确显示采购负责人
7. **经理进度跟踪**：查看经理视角的合同详情是否显示负责人
8. **经理关联对比**：查看关联对比页面是否显示负责人
