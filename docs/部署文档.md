### 1.4 遇到的问题及解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| Go 版本不匹配 | go.mod 需要 1.24+ | 使用 golang:1.24-alpine |
| Go 模块下载超时 | 网络墙 | 添加 GOPROXY=https://goproxy.cn,direct |
| npm install 超时 | 网络墙 | 添加 npm registry=https://registry.npmmirror.com |
| 端口 80 被占用 | frontend 直接暴露端口 | 移除 frontend 的 ports 配置 |
| Traefik 无法连接 Docker | API 版本不兼容 | 使用静态配置文件替代 docker provider |
| 数据丢失 | 未挂载本地目录 | 将 ./backend/pb_data 挂载到容器 |

### 2.3 部署步骤

#### 步骤 1：服务器环境准备

```bash
# 1. 安装 Docker
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER

# 2. 安装 Docker Compose
sudo curl -L "https://github.com/docker/compose/releases/download/v2.24.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
sudo chmod +x /usr/local/bin/docker-compose

# 3. 验证安装
docker --version
docker-compose --version
```

#### 步骤 2：上传项目文件

```bash
# 在本地打包
tar -czvf erp-system.tar.gz \
  docker-compose.yml \
  frontend/ \
  backend/ \
  traefik/

# 上传到服务器
scp erp-system.tar.gz root@182.92.78.227:/root/

# 在服务器解压
ssh root@182.92.78.227
cd /root
tar -xzvf erp-system.tar.gz
```

#### 步骤 3：修改生产环境配置

1. **更新 docker-compose.yml**：移除调试配置

2. **配置 Traefik 支持 HTTPS**：

创建 `traefik/tls.toml`：
```toml
[tls]
  [[tls.certificates]]
    certFile = "/certs/henghuacheng.cn.pem"
    keyFile = "/certs/henghuacheng.cn.key"
```

3. **获取 SSL 证书**（推荐使用 Let's Encrypt）：

```bash
# 使用 certbot 获取免费证书（需先配置域名解析）
sudo apt install certbot
sudo certbot certonly --webroot -w /var/www/html -d henghuacheng.cn -d api.henghuacheng.cn

# 证书位置
# /etc/letsencrypt/live/henghuacheng.cn/fullchain.pem
# /etc/letsencrypt/live/henghuacheng.cn/privkey.pem
```

#### 步骤 4：修改 Traefik 配置

更新 `traefik/traefik.yml`：
```yaml
global:
  checkNewVersion: false
  sendAnonymousUsage: false

api:
  insecure: true

entryPoints:
  web:
    address: ":80"
  websecure:
    address: ":443"

providers:
  file:
    filename: /etc/traefik/dynamic.toml

certificatesResolvers:
  letsencrypt:
    acme:
      email: your-email@example.com
      storage: /etc/traefik/acme.json
      httpChallenge:
        entryPoint: web
```

更新 `traefik/dynamic.toml`：
```toml
[http.routers]
  [http.routers.frontend]
    rule = "Host(`henghuacheng.cn`)"
    service = "frontend"
    entryPoints = ["web", "websecure"]
    [http.routers.frontend.tls]
      certResolver = "letsencrypt"

  [http.routers.pocketbase]
    rule = "Host(`api.henghuacheng.cn`)"
    service = "pocketbase"
    entryPoints = ["web", "websecure"]
    [http.routers.pocketbase.tls]
      certResolver = "letsencrypt"

[http.services]
  [http.services.frontend]
    [[http.services.frontend.loadBalancer.servers]]
      url = "http://my-erp-system-frontend-1:80"

  [http.services.pocketbase]
    [[http.services.pocketbase.loadBalancer.servers]]
      url = "http://my-erp-system-pocketbase-1:8090"
```

#### 步骤 5：启动服务

```bash
cd /root/erp-system
docker-compose up -d
```

#### 步骤 6：验证部署

|------|------|

### 3.1 防火墙配置

服务器需开放以下端口：

| 端口 | 协议 | 用途 |
|------|------|------|
| 80 | TCP | HTTP |
| 443 | TCP | HTTPS |
| 8080 | TCP | Traefik Dashboard（可选） |

配置命令：
```bash
# ufw 方式
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp
sudo ufw allow 8080/tcp
sudo ufw reload
```

同时在阿里云安全组添加入方向规则：端口 80/443/8080，授权对象 0.0.0.0/0
