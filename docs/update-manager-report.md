# 合同金额（含税）修改说明

## 背景说明

上传的销售合同和采购合同中的合同金额（原来版本的"销售总价"、"采购总价"）都是**含税金额**（即含增值税的金额）。

---

## 修改内容

### 一、关联对比中的利润分析公式修改

#### 1. 税额计算
```
税额 = (销售合同上写的总价（含税） - 采购合同上写的总价（含税）) × 0.1881
```

#### 2. 净利润计算
```
净利润 = 销售合同上写的总价（含税）/ 1.13 - 采购合同上写的总价（含税）/ 1.13 - 税额 - 杂费及运费
```

#### 3. 不含税金额换算（已知含税金额，求不含税金额）
```
销售总价（不含税）= 销售合同上写的总价（含税）÷ 1.13
采购总价（不含税）= 采购合同上写的总价（含税）÷ 1.13
```

---

### 二、数据报表修改

#### 1. 字段调整
- **修改字段1**：销售含税总价、采购含税总价直接把销售合同和采购合同上的合同总金额填写上去
- **修改字段2**：销售总价（不含税）、采购总价（不含税）现在需要用上面那个公式，
  销售总价（不含税）= 销售合同上写的总价（含税）÷ 1.13
  采购总价（不含税）= 采购合同上写的总价（含税）÷ 1.13

#### 2. 税额和净利润
- 税额和净利润的计算方式与关联对比中的利润分析一致，直接复用上述利润分析中计算公式

---

## 计算示例

假设：
- 销售合同含税总价：113,000 元（合同上传的，在合同页面应该叫总金额，后端字段叫total_amount）
- 采购合同含税总价：56,500 元（合同上传的，在合同页面应该叫总金额，后端字段叫total_amount）
- 杂费及运费：1,000 元

计算过程：
1. 销售不含税总价 = 113,000 ÷ 1.13 = 100,000 元
2. 采购不含税总价 = 56,500 ÷ 1.13 = 50,000 元
3. 税额 = (113,000 - 56,500) × 0.1881 = 10,627.35 元
4. 净利润 = 100,000 - 50,000 - 10,627.35 - 1,000 = 38,372.65 元
