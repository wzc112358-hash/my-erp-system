# 企业采购销售管理系统 — 使用手册

---

## 一、登录与系统选择

### 1.1 选择系统

首次打开系统会进入系统选择页面，请根据您的业务选择**北京系统**或**兰州系统**。

> 如果选错了，清除浏览器缓存后重新访问即可重新选择。

### 1.2 登录

使用分配给您的账号密码登录。登录后根据您的角色自动进入对应模块：
- **销售职员** → 进入销售模块
- **采购职员** → 进入采购模块
- **经理** → 进入经理模块

---

## 二、销售模块

### 2.1 客户管理

**路径**：侧边栏 → 客户管理

- 点击「新建客户」填写客户名称、联系人、电话等信息
- 点击客户名称可查看该客户的所有关联合同和交易记录

### 2.2 销售合同

**路径**：侧边栏 → 销售合同

#### 新建合同

1. 点击「新建合同」
2. 填写合同信息：
   - **合同编号**：手动输入
   - **客户**：从下拉列表选择
   - **品名、单价、数量**：按实际填写
   - **签约日期**：选择日期
3. 根据实际情况选择开关：
   - **「按不含税单价填写」**：如果您的单价是不含税的，请打开此开关。打开后标签会变为「产品单价（不含税）」
   - **「跨境交易（USD）」**：如果是境外客户用美元结算，请打开此开关。打开后金额会以美元显示，并自动折算人民币
4. 可上传合同附件（支持多个文件）
5. 点击「提交」

> 合同提交后，系统会**自动通知所有采购人员**，他们可以直接从通知中创建关联的采购合同。

#### 查看合同

在合同列表中点击合同编号进入详情页，可以查看：

- **合同基本信息**：编号、客户、品名、金额、数量、状态等
- **三个进度卡片**（可点击跳转）：
  - 发货进度：已发货数量 / 合同总数量
  - 收款进度：已收金额 / 合同总金额
  - 开票进度：已开票金额 / 合同总金额
- **发货批次表格**：每批次发货的详细信息
- **发票记录表格**：每张开票的详细信息
- **收款记录表格**：每笔收款的详细信息
- **欠款信息**：当前欠款金额和比例

#### 如果是跨境合同

详情页中：
- 金额显示为 `$美元金额（≈ ¥人民币金额）` 格式
- 侧边栏底部显示当前汇率

### 2.3 运输（发货）

**路径**：侧边栏 → 运输

1. 点击「新建发货」
2. 选择关联的销售合同
3. 填写运单号、发货日期、发货数量、物流公司、送货地址等
4. 提交后，该合同的发货进度会自动更新

> 如果关联合同是跨境的，合同金额会同时显示美元和人民币。

### 2.4 发票

**路径**：侧边栏 → 发票

1. 点击「新建发票」
2. 选择关联的销售合同
3. 填写发票号、发票类型、产品金额、发票金额、开票日期
4. 提交后进入**待经理确认**状态
5. 经理确认后您会收到通知

### 2.5 收款

**路径**：侧边栏 → 收款

1. 点击「新建收款」
2. 选择关联的销售合同
3. 填写收款金额、收款日期、收款方式等
4. 提交后进入**待经理确认**状态
5. 经理确认后您会收到通知

### 2.6 服务合同 & 投标管理

- **服务合同**：管理服务类合同，填写客户、品名、单价、金额等信息
- **投标管理**：记录投标信息，包括标书费、投标保证金、招标代理费等

### 2.7 通知中心

**路径**：侧边栏 → 通知中心

您会收到以下通知：

| 通知 | 说明 |
|------|------|
| 采购合同已创建 | 采购人员创建了关联您销售合同的采购合同 |
| 经理已确认 | 经理确认了您的开票/收款申请 |
| 汇率已变更 | 经理修改了美元汇率 |

**快速操作**：收到采购合同通知后，可以点击「去创建销售合同」直接跳转并自动关联。

### 2.8 汇率设置

**路径**：侧边栏 → 汇率设置

查看当前美元兑人民币汇率。如需修改请联系经理操作。

## 三、采购模块

### 3.1 供应商管理

**路径**：侧边栏 → 供应商管理

- 点击「新建供应商」填写供应商名称、联系人、电话等信息
- 点击供应商名称可查看该供应商的所有关联合同和交易记录

### 3.2 采购合同

**路径**：侧边栏 → 采购合同

#### 从通知创建（推荐）

1. 进入「通知中心」，找到销售合同通知
2. 点击通知查看详情
3. 点击「去创建采购合同」按钮
4. 系统自动关联对应的销售合同，填写采购信息即可

#### 手动创建

1. 点击「新建合同」
2. 填写合同信息：
   - **合同编号**：手动输入
   - **供应商**：从下拉列表选择
   - **关联销售合同**：选择要关联的销售合同
   - **品名、单价、数量**：按实际填写
   - **签约日期**：选择日期
3. **「跨境交易（USD）」**：如果供应商在境外用美元结算，请打开此开关
4. 上传合同附件
5. 点击「提交」

> 关联销售合同后，系统会**自动通知对应的销售人员**。

#### 查看合同

在合同列表中点击合同编号进入详情页：
- **合同基本信息**：编号、供应商、品名、金额、数量等
- **三个进度卡片**：
  - 到货进度
  - 开票进度
  - 付款进度
- **关联销售合同**：可查看销售合同的基本信息（**不含单价和总金额**）

### 3.3 运输（到货）

**路径**：侧边栏 → 运输

1. 点击「新建到货」
2. 选择关联的采购合同
3. 填写运单号、发货日期、到货数量、物流公司
4. 如有运费、杂费，一并填写
5. 提交后进入**待经理确认**状态

### 3.4 收票

**路径**：侧边栏 → 收票

1. 点击「新建收票」
2. 选择关联的采购合同
3. 填写发票号、发票类型、金额、收票日期
4. 提交后进入**待经理确认**状态

### 3.5 付款

**路径**：侧边栏 → 付款

1. 点击「新建付款」
2. 选择关联的采购合同
3. 填写付款金额、付款日期、付款方式
4. 提交后进入**待经理确认**状态

### 3.6 资金支出

**路径**：侧边栏 → 资金支出

记录与合同无关的其他资金支出，填写支出金额、付款金额、付款日期等。

### 3.7 通知中心

您会收到以下通知：

| 通知 | 说明 |
|------|------|
| 销售合同已创建 | 有新的销售合同需要采购 |
| 经理已确认 | 经理确认了您的到货/收票/付款申请 |
| 汇率已变更 | 经理修改了美元汇率 |

### 3.8 汇率设置

**路径**：侧边栏 → 汇率设置

查看当前美元兑人民币汇率。如需修改请联系经理操作。

## 四、管理模块

### 4.1 关联合同总览

**路径**：侧边栏 → 关联合同总览

**功能**：一目了然地查看所有销售合同和对应采购合同的关联关系。

- **左侧**：销售合同列表
- **右侧**：对应关联的采购合同汇总

**操作说明**：
- 点击左侧销售合同卡片 → 进入合同详情页
- 点击右侧采购汇总卡片 → 弹出窗口查看每个采购合同的详情
- 勾选合同后点击「导出报表」→ 只导出选中合同的数据
- 支持按客户、品名、时间段筛选

### 4.2 合同详情页

从总览页点击销售合同进入，展示完整的合同信息。

页面分为两个标签页：

**销售合同信息**：
- 销售合同基本信息（可上传附件）
- 发货批次列表（点击展开可查看附件）
- 发票记录列表
- 收款记录列表

**采购合同信息**：
- 所有关联采购合同的基本信息
- 到货批次列表
- 收票记录列表
- 付款记录列表

**经理确认操作**：
- 每条记录旁边显示确认状态：🟡待确认 / 🟢已确认 / 🔴已驳回
- 点击「确认」或「驳回」按钮操作
- 操作后系统**自动通知**对应的销售/采购人员

### 4.3 利润分析

在合同详情页底部，自动计算利润。

**显示内容**：
- 销售总金额（含税/不含税）
- 采购总金额（含税/不含税）
- 运费合计
- 杂费合计
- 营业利润
- 税额
- 净利润

**跨境合同双币种分析**：

当销售合同和所有采购合同都是跨境交易时，利润分析会显示两个标签页：
- **人民币分析**：所有金额按汇率换算为人民币后计算利润
- **美元分析**：直接用美元金额计算利润

两个分析使用相同的计算公式，方便您从两个角度评估利润。

**计算公式**：
- 营业利润 = 销售金额 - 采购金额 - 运费 - 杂费
- 税额 = (销售金额 - 采购金额) × 18.81%
- 净利润 = 营业利润 - 税额

### 4.4 流程进度

**路径**：侧边栏 → 流程进度

**功能**：以流程图形式查看每个合同的执行进度和待确认事项。

**操作**：
1. 下拉选择一个销售合同
2. 流程图从左到右展示完整流程：
   - 销售合同 → 发货 → 开票 → 收款
   - 采购合同 → 到货 → 收票 → 付款
3. 待确认的节点以**红色虚线边框**标注
4. 点击节点可查看详情并执行确认操作

> 侧边栏「流程进度」菜单旁的**红色数字**表示当前有多少条待确认记录。

### 4.5 数据报表

**路径**：侧边栏 → 数据报表

**功能**：按时间段查看所有合同的利润报表。

**操作**：
- 选择年份和月份范围
- 点击「查询」生成报表
- 点击「导出 Excel」下载报表文件
- 点击表格行可跳转到对应合同详情

**从总览页导出**：在总览页勾选合同后点击「导出报表」，报表页只显示选中合同的数据。点击「清除筛选」恢复查看全部。

### 4.6 业绩统计

**路径**：侧边栏 → 业绩统计

**功能**：按业务员统计合同数量和总金额。

三个标签页：
- **销售业绩**：每位销售人员的合同数和金额
- **采购业绩**：每位采购人员的合同数和金额
- **佣金业绩**：服务合同的业绩

支持按日期范围筛选。点击某一行可查看该业务员的合同明细。

### 4.7 其他业务

**路径**：侧边栏 → 其他业务

管理服务合同、投标记录和资金支出等非主营业务。

### 4.8 汇率设置

**路径**：侧边栏 → 汇率设置

**功能**：查看和修改美元兑人民币汇率。

**操作**：
1. 页面显示当前汇率
2. 输入新的汇率值
3. 点击「保存并通知相关人员」
4. 系统自动通知所有销售和采购人员

> 汇率修改后，所有跨境合同的折算金额会自动更新。
